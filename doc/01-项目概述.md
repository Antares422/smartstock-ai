# 智能股票交易辅助系统 - 项目概述

**版本**: v0.0.1-SNAPSHOT
**最后更新**: 2026-03-09

---

## 1. 项目简介

### 1.1 项目定位

**smartstock-ai** 是一个基于 AI 技术的智能股票交易辅助系统，旨在为个人投资者和量化交易爱好者提供实时行情监控、智能分析、模拟交易和策略回测等一站式服务。

### 1.2 核心价值主张

- **实时智能**：毫秒级行情推送 + AI 实时解读，让投资决策更快更准
- **风险可控**：多维度风险预警 + 智能止损止盈，保护投资本金
- **策略驱动**：自定义策略回测 + AI 参数优化，提升交易胜率
- **模拟先行**：零风险模拟交易环境，验证策略后再实盘操作

### 1.3 目标用户群体

- **个人投资者**：希望通过 AI 辅助提升投资决策质量
- **量化交易爱好者**：需要策略回测和参数优化工具
- **金融科技学习者**：学习 AI + 金融科技的实战项目

## 2. 核心功能概览

### 2.1 实时行情监控

接入主流股票行情 API，通过 Kafka 处理高并发数据流，Redis 缓存热点数据，WebSocket 实时推送到前端。支持自选股管理、多维度行情展示（K 线图、分时图、成交量等）。

### 2.2 AI 智能分析

- **技术指标分析**：自动计算 MACD、KDJ、RSI 等常用技术指标
- **AI 行情解读**：接入 Claude API，对行情数据进行自然语言解读
- **新闻情绪分析**：抓取财经新闻，分析市场情绪对股价的影响
- **智能问答**：用户可以自然语言提问，AI 给出投资建议

### 2.3 模拟交易

提供虚拟账户和虚拟资金，支持买入、卖出、持仓管理、交易记录查询、盈亏统计等功能。所有交易基于真实行情数据，但不涉及真实资金，适合策略验证和学习。

### 2.4 智能策略回测

用户可以自定义交易策略（如均线策略、突破策略等），系统基于历史数据进行回测，生成收益曲线、最大回撤、夏普比率等指标。AI 可以根据回测结果优化策略参数。

### 2.5 风险预警

- **持仓风险监控**：实时计算持仓集中度、行业分布等风险指标
- **价格波动预测**：基于历史数据和 AI 模型预测短期价格波动
- **止损止盈提醒**：自动触发止损止盈提醒，避免重大损失

## 4. 系统架构概览

### 4.1 架构分层

系统采用经典的三层架构：

- **表现层（Presentation Layer）**：Vue 3 前端 + WebSocket 实时通信
- **业务逻辑层（Business Logic Layer）**：Spring Boot 后端服务
- **数据访问层（Data Access Layer）**：MyBatis Plus + MySQL

### 4.2 核心模块

```
smartstock-ai
├── market-service（行情服务）
│   ├── 行情数据接入
│   ├── Kafka 数据流处理
│   └── WebSocket 推送
├── ai-service（AI 服务）
│   ├── 技术指标计算
│   ├── Claude API 集成
│   └── 新闻情绪分析
├── trade-service（交易服务）
│   ├── 模拟账户管理
│   ├── 买卖订单处理
│   └── 持仓盈亏计算
├── backtest-service（回测服务）
│   ├── 策略引擎
│   ├── 历史数据回放
│   └── 回测报告生成
└── risk-service（风险服务）
    ├── 风险指标计算
    ├── 预警规则引擎
    └── 消息通知
```

详细的架构设计请参考 [03-技术架构文档.md](./03-技术架构文档.md)。

## 5. 快速开始

### 5.1 环境要求

| 软件 | 版本要求 |
|------|----------|
| JDK | 17+ |
| Maven | 3.6+ |
| MySQL | 8.0+ |
| Redis | 7.0+ |
| Kafka | 3.0+ |
| MinIO | Latest |

### 5.2 快速启动步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd smartstock-ai
   ```

2. **配置数据库**
   ```bash
   # 创建数据库
   mysql -u root -p785460 -e "CREATE DATABASE smartstock_ai DEFAULT CHARACTER SET utf8mb4;"

   # 导入初始化脚本（待补充）
   mysql -u root -p785460 smartstock_ai < doc/sql/init.sql
   ```

3. **配置中间件**
   ```bash
   # 启动 Redis
   systemctl start redis-server

   # 启动 Kafka
   systemctl start kafka

   # 启动 MinIO
   systemctl start minio
   ```

4. **修改配置文件**
   ```bash
   # 编辑 src/main/resources/application.yml
   # 配置数据库连接、Redis、Kafka、MinIO 等
   ```

5. **启动后端服务**
   ```bash
   mvn clean install
   mvn spring-boot:run
   ```

6. **启动前端服务**（待补充）
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

详细的开发环境搭建和部署指南请参考 [06-开发指南.md](./06-开发指南.md)。
